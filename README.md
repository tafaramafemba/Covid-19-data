# Math Magicians
This is a mobile application that tracks Covid-19 metrics.

## Built with:
 * Reactjs
 * Redux

## Usage
To clone the repo:

$ git clone git@github.com:clone tafaramafemba/covid-19-data.git


To build the project run:

$ npm install

$ npm run build

## Development

In development environment use the following command:

$ npm start
